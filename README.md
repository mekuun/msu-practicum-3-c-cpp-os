# Практикум 3 семестра: C, C++ и операционные системы

## Структура

В репозитории собраны задачи по практикуму за 3 семестр: C, C++ и задачи по операционным системам.
2023-2024 уч.г.

```text
.
├── practicum
│   ├── c-cpp-small-tasks
│   ├── c-cpp-homeworks
│   ├── os-small-tasks
│   ├── os-homeworks
│   ├── control-work
│   └── processes-and-signals
└── operating-systems
    └── multithreading-and-ipc
```

## Содержание

### `practicum/c-cpp-small-tasks`

Малые задачи по C и C++.

### `practicum/c-cpp-homeworks`

Домашние задания по C и C++.

### `practicum/os-small-tasks`

Малые задачи по операционным системам из практикума.

### `practicum/os-homeworks`

Домашние задания по операционным системам.

### `practicum/control-work`

Контрольные и задачи в формате проверяющей системы.

### `practicum/processes-and-signals`

Задачи на процессы, сигналы, семафоры, массивы и другие низкоуровневые темы.

### `operating-systems/multithreading-and-ipc`

Практика по операционным системам: потоки, мьютексы, каналы, сигналы, `fork`/`exec` и синхронизация читателей-писателей.
